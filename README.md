# Vattenfall Week 9 Capstone

## Project Theme

Vattenfall — Energy Operations and Market Intelligence Lakehouse

This project simulates a real-world Databricks-based data engineering solution. It integrates multiple data sources such as energy market prices, weather data, grid telemetry, and incident events to build a governed lakehouse architecture.

The goal is to transform raw data into structured, analytics-ready datasets through bronze, silver, and gold layers, ultimately supporting reporting and business insights.

## Business Domains

* Energy Market Intelligence
* Data Engineering & Analytics
* Sustainability & Grid Operations

## Day 1 Focus

Day 1 is focused on building the engineering foundation of the project. This includes:

* Defining project structure and repository layout

* Organizing raw data sources
* Designing a governed target structure
* Setting up initial documentation and configuration approach
