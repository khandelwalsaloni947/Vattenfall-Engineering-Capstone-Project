# How to Use This Repository

This repository is structured to simulate a real-world data engineering project.

## Navigation Guide

* `docs/` → Project documentation and design decisions
* `config/` → Configuration files
* `sample_data/` → Raw and sample datasets
* `sql/` → SQL transformations
* `src/` → Core Python logic
* `notebooks/` → Data processing notebooks
* `tests/` → Testing logic

## Getting Started

1. Read the business context
2. Review the architecture overview
3. Explore the repository structure
4. Follow the Day 1 scope and deliverables

This structure ensures clarity, scalability, and maintainability.
